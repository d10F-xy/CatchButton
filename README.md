# ★버튼 잡기 게임★ (C# 코딩)

## 개요 
- C# 프로그래밍 학습
- 핵심 기능 : 마우스를 버튼 위에 올려두면 버튼이 도망
- 화면 구성 : 폼 제목에는 현재 버튼의 X, Y 값을 표시하고 폼 화면에는 도망다니는 버튼이 존재함

# 구현 시 어려웠던 점
- 처음에 버튼이 폼을 이탈하는 경우가 발생해 해결하는데 고민을 하였음
- 소리를 구현하는 명령어가 무엇인지 몰라 찾아 헤맸지만 시스템 파일을 사용하여 해결하였음
- 버튼을 잡았을 경우 성공 메시지를 띄우는 과정은 메시지박스로 구현하였음
- 버튼의 크기가 감소는 Width 와 Height 의 수정으로 해결함
- 결과적으로 2주차 수업을 들으며 어려운 부분이 있었지만 하나씩 해결해 나가면서 보람찬 수업이였던 것 같음
